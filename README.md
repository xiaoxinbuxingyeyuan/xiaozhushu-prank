# 懂猪帝

> 懂不懂小猪都用懂猪帝。

懂猪帝是一个带有“小猪恶作剧”气质的日常记录网站。它最初来自一个仿小红书信息流的小实验：把满屏帖子换成小猪，再把朋友的照片悄悄混入其中。后来它逐渐升级成一个可以创建个人小社区、邀请朋友一起发布日常、评论、点赞和回看回忆日历的小型社交网站。

## 项目特点

- 原生小猪示例区：新用户可以先浏览系统准备的小猪示例帖。
- 个人帖子社区：用户登录后可以创建自己的“小猪社区”。
- 邀请好友加入：通过邀请链接让朋友进入同一个社区。
- 日常记录发布：支持纯文字、图片、视频、标签和可选标题。
- 评论与点赞：社区成员可以互动、评论和点赞。
- 回忆日历：用日历方式查看某一天发布过哪些内容。
- 账号体系：支持邮箱密码登录、注册、退出账号和忘记密码重置。
- 管理台：支持帖子管理、举报处理和基础容量提示。
- 日记胶片风 UI：整体采用温暖、轻松、带一点贴纸和恶作剧感的视觉风格。

## 技术栈

- 前端：HTML、CSS、Vanilla JavaScript
- 托管：GitHub Pages
- 后端：Supabase
  - Auth：邮箱密码登录、密码重置
  - Database：Postgres + RLS 权限控制
  - Storage：用户上传媒体文件
  - Realtime：评论、点赞和帖子更新

## 当前功能状态

目前项目已经支持：

- 浏览小猪示例帖
- 注册 / 登录 / 退出账号
- 忘记密码，通过邮箱链接或验证码重置
- 创建个人社区
- 通过邀请链接加入社区
- 在社区内发布帖子
- 支持纯文字帖子
- 支持图片和短视频上传
- 评论、回复、点赞和收藏
- 按日期回看帖子
- 管理员查看举报、管理帖子和查看容量估算

## 部署方式

项目是静态前端，无需构建步骤。整个目录可以直接部署到 GitHub Pages、Cloudflare Pages 或其他静态托管服务。

当前正式部署使用：

- GitHub Pages 负责前端页面
- Supabase 负责账号、数据库、评论、点赞、社区和媒体存储

## 项目定位

懂猪帝不是一个严肃的大型社交平台，而是一个偏私人、轻量、好玩的日常记录空间。

它适合用来：

- 和朋友一起记录旅行、聚会、校园或生活碎片
- 做一个带彩蛋的生日/恶作剧小网站
- 搭建小圈子的照片墙和评论区
- 练习完整的前端 + Supabase 小型产品开发流程
